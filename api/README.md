# Installation

```bash
npm install
```

# Démarrage

```bash
npm start
```

# Fichiers à créer
## Fichier de configuration API

*/api/api_conf.json*

```json
{
    "api_uri": "elurldelapi"
}
```

## Fichier de configuration documentation API (Swagger)

*/api/swagger/swagger.json*
